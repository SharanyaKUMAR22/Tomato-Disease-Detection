Crop disease detection using ML

The objective of this project is to develop an accurate and explainable deep learning model capable of automatically detecting and classifying tomato leaf diseases from images.

Problem Statement

Tomato leaf diseases cause major agricultural losses, and manual detection is slow and error-prone. Existing AI models often lack interpretability, making them unreliable for practical use. This project aims to develop an accurate and explainable disease detection system to support early and reliable crop diagnosis.

Literature Review / Market Research

Existing research on tomato leaf disease detection primarily uses machine learning and deep learning models such as SVM, ResNet, VGG, and MobileNet. While these models achieve high accuracy on benchmark datasets like PlantVillage, they often rely on controlled environments and lack explainability. Most systems do not provide visual interpretation of predictions, limiting their practical adoption in real-world agricultural scenarios.

Research Gap / Innovation

My approach improves upon existing methods by combining EfficientNet-based transfer learning with Grad-CAM explainability. Unlike many state-of-the-art black-box models, my system provides visual evidence of disease-affected regions, enhancing trust and practical usability. Additionally, I focus on confidence calibration and deployment-readiness, making the system more suitable for real-world agricultural applications.

System Methodology

Dataset / Input

Dataset / Input

  • Source: PlantVillage Dataset (Tomato leaf subset).
  • Size: Multiple disease classes with thousands of labeled images.
  • Preprocessing:
    • Images resized to 224 × 224 pixels.
    • Normalized using EfficientNet preprocessing.
    • Data augmentation applied to improve generalization.

Model / Architecture

Algorithms Used / System Workflow

  • Transfer Learning (EfficientNet-B0): Pre-trained EfficientNet-B0 model used for feature extraction and classification.
  • Image Preprocessing: Images resized to 224 × 224 pixels and normalized using EfficientNet preprocessing.
  • Model Training: Fine-tuning performed on tomato leaf dataset using categorical cross-entropy loss and Adam optimizer.
  • Prediction Pipeline: Uploaded image → Preprocessing → Model Inference → Class Prediction → Confidence Score.
  • Explainability (Grad-CAM): Grad-CAM visualization highlights disease-affected regions on the leaf to improve interpretability.
